Authentication

To use this service, your gpg public key must be part of my key-ring.

What?

Generate a gpg key if you haven't gpg --full-generate-key

Upload your gpg key to a keyserver, preferrably this one

Then provide me with the e-mail your key is registered with so I can fetch it from the keyserver

All files that are uploaded must be digitally signed (see example below)

Example upload: 

    # create tmp file
    TMPFILE='/tmp/hostmystuff.tmp';
    # sign the file that you wish to upload 
    SIGNATURE=$(gpg --sign --output $TMPFILE 'path-to-file');
    # do the request
    LINK=$(curl -F"file=@$TMPFILE" https://www.hostmystuff.ml)
    # remove the tmp file
    $(rm $TMPFILE)
    # put the link to the file in stdout
    echo "$LINK"

And yes.. it will ask you for your passphrase sometimes when you upload, deal with it!

Syntax highlighting

HostMyStuff supports syntax highlighting. See this hello world programme with some highlight sugar

public class HelloWorld {
	public static void main(String... args) {
		System.out.println("Hello World!");	
	}	
}

To not apply syntax highlighting simply supply a raw query parameter to the GET request ?raw=true. Useful in-case of a bug for highlighting is applied (please report them if you find any) or in case you want to download the file (e.g. wget).

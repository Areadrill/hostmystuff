Welcome to HostMyStuff

To use this service, you must ask for whoever is hosting it for an api key to use it. Unless, of course, they have disabled authorization.

Example upload: curl --header 'key: your-api-key' -F'file=@file-name' host-of-this-website

Some mime-types are not allowed:

  • 'application/x-dosexec'
  • 'application/x-executable'
  • 'application/vnd.android.package-archive'

Files get deleted after 24 hours!

Welcome to HostMyStuff

To use this service, you must get an api key.

Example upload: curl --header 'key: your-api-key' -F'file=@file-name' https://www.hostmystuff.ml

Max file size is 512M

Files get deleted after 13 hours!

Syntax highlighting

HostMyStuff supports syntax highlighting. See this hello world programme with some highlight sugar

public class HelloWorld {
	public static void main(String... args) {
		System.out.println("Hello World!");	
	}	
}

To not apply syntax highlighting simply supply a raw query parameter to the GET request ?raw=true. Useful in-case of a bug for highlighting is applied (please report them if you find any) or in case you want to download the file (e.g. wget).
